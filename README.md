Auth System Project

This is a simple authentication system with a backend and frontend.

Getting Started

 1. Clone the repository

    git clone https://github.com/souveniriranzi/auth-system.git
    cd auth-system
2. Backend Setup
  cd backend
  npm install
To start the backend server:

   npm start
3. Frontend Setup
   cd frontend
  npm install

To start the frontend server:

  npm start
